# DataSpark: Illuminating Insights for Global Electronics

## üõçÔ∏è Domain
**Retail Analytics in the Electronics Industry**

## üöÄ Skills and Technologies
- **Skills**: Data Cleaning and Preprocessing, Exploratory Data Analysis (EDA), Python, Data Management using SQL, Power BI for visualizations
- **Technologies**: Python 3.9, MySQL 8.0, Power BI, Matplotlib, Seaborn

## üìä Business Use Cases
Analyze customer, product, sales, and store data to:
- Enhance marketing strategies with data-driven insights.
- Optimize inventory management through product and sales performance analysis.
- Improve sales forecasting by identifying trends and seasonality.
- Develop pricing strategies based on the impact of currency exchange rates.

These insights will help Global Electronics improve customer satisfaction, streamline operations, and drive business growth.

## üìà Approach
1. **Data Cleaning and Preparation**: Addressed missing values, ensured accurate data types, and merged multiple datasets (e.g., sales with product and customer data).
2. **Data Loading**: Preprocessed data was inserted into SQL databases for efficient querying and analysis.
3. **SQL Queries**: Developed 10 SQL queries to extract insights into customer demographics, product performance, and store operations.
4. **Power BI Visualization**: Created interactive dashboards using Power BI to visualize sales trends, product popularity, and store performance.
